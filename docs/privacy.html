Italiano

Sintesi

TechSnap registra il tab attivo solo dopo azione esplicita dell'utente (Start). L'estensione e progettata per elaborazione locale ed export locale.

Dati trattati

  • Titolo e URL del tab attivo.
  • Stream audio/video del tab selezionato durante la registrazione.
  • Testo sottotitoli inserito dall'utente (tastiera o dettatura).
  • Stato temporaneo del recorder in chrome.storage.session.

Uso dei dati

  • Registrazione tutorial ed export file .webm, .vtt, .srt, .html.
  • Visualizzazione metadati (titolo/url) nell'interfaccia e nei file esportati.

Condivisione e trasferimenti

  • Nessun backend proprietario.
  • Nessun invio dati ai server dello sviluppatore.
  • I dati restano nel browser e nella cartella download dell'utente, salvo i servizi piattaforma usati dalle API browser.

Voce e conservazione

  • La modalita voce usa Web Speech API: il browser/sistema puo elaborare audio microfono.
  • Lo stato sessione e temporaneo; i file sono salvati solo su richiesta utente.
  • La rimozione estensione elimina i dati locali gestiti da Chrome.

English

Summary

TechSnap records the active tab only after explicit user action (Start). The extension is designed for local processing and local export.

Data processed

  • Active tab title and URL.
  • Audio/video stream of the selected tab during recording.
  • Subtitle text entered by the user (keyboard or voice dictation).
  • Temporary recorder state in chrome.storage.session.

How data is used

  • Run core functionality and export .webm, .vtt, .srt, and preview .html files.
  • Show recording metadata (title/url) in the recorder UI and exports.

Data sharing and transfer

  • No custom backend is used.
  • No user data is sent to the developer's servers.
  • Data stays in browser context and local downloads, except browser/platform services behind built-in APIs.

Voice and retention

  • Voice mode uses Web Speech API; microphone audio may be processed by browser/platform speech services.
  • Session state is temporary; files are saved only on explicit user request.
  • Removing the extension removes extension-local data managed by Chrome.
