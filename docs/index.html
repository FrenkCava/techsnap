Italiano

TechSnap ti permette di registrare il tab attivo e creare sottotitoli sincronizzati durante la registrazione.

Installazione (utente finale)

  1. Apri la pagina dell'estensione sul Chrome Web Store.
  2. Clicca Aggiungi a Chrome.
  3. Conferma con Aggiungi estensione.
  4. Fissa l'icona di TechSnap dalla barra estensioni per accesso rapido.

Come si usa

  1. Apri la pagina/tab che vuoi registrare.
  2. Clicca l'icona TechSnap e si apre il pannello recorder.
  3. Premi Avvia per iniziare la cattura del tab attivo.
  4. Durante la registrazione puoi usare Pausa e Riprendi.
  5. Premi Stop per terminare.
  6. Scarica video e sottotitoli nei formati disponibili.

Sottotitoli - modalita testuale

  • Seleziona la lingua attiva (IT/EN/FR/DE/ES/PT/ZH/JA).
  • Opzionale: premi F2 o "Segna Inizio" per marcare il punto iniziale.
  • Scrivi il testo e premi "Aggiungi Sottotitolo" (oppure Ctrl+Enter).

Sottotitoli - modalita vocale

  • Passa alla modalita voce e abilita il microfono quando richiesto.
  • Premi "Ascolta", parla, poi conferma con "Aggiungi Testo".
  • La lingua del riconoscimento segue la lingua attiva selezionata.

Export disponibili

  • Video .webm.
  • Sottotitoli .vtt e .srt per ogni lingua con contenuti.
  • Preview .html con player e tracce sottotitoli.
ffmpeg -i tutorial.webm -c:v libx264 -c:a aac tutorial.mp4

English

TechSnap lets you record the active tab and create synchronized subtitles while recording.

Installation (end users)

  1. Open the extension page on the Chrome Web Store.
  2. Click Add to Chrome.
  3. Confirm with Add extension.
  4. Pin the TechSnap icon from the extensions menu for quick access.

How to use

  1. Open the tab you want to record.
  2. Click the TechSnap icon to open the recorder panel.
  3. Press Start to begin active-tab capture.
  4. Use Pause and Resume as needed.
  5. Press Stop to finish.
  6. Download video and subtitle exports.

Subtitles - text mode

  • Select the active subtitle language (IT/EN/FR/DE/ES/PT/ZH/JA).
  • Optional: press F2 or "Mark Start" to set subtitle start time.
  • Type text and click "Add Subtitle" (or use Ctrl+Enter).

Subtitles - voice mode

  • Switch to voice mode and allow microphone access when prompted.
  • Click "Listen", speak, then confirm with "Add Text".
  • Speech recognition language follows the currently selected subtitle language.

Available exports

  • .webm video.
  • .vtt and .srt subtitles per language.
  • .html preview page with player and subtitle tracks.
